EXPRESSIVE SPEECH SYNTHESIS INDONESIAN LANGUANGE WITH EXTRACTION F0 USING INSTANTANEOUS FREQUENCY AMPLITUDE SPECTRUM (IFAS)

Indonesian Emotion Speech Samples generated from Speech sythesis. The synthetic speeches are generated with different System and different Vocoder. Using Indonesian Expressive speech database[1] to train model for Speech Sythesis Model.

This page contains following samples:

  • Original Speech
  • HTS with F0 IFAS[2]: F0 values are genereted using IFAS, trained with 227 Happiness Voice, 213 Anger Voice, 215 Sadness Voice. Source: HTS
  • HTS with STRAIGHT Vocoder[3]: Acoustic Feature are generated and sythesis using STRAIGHT Vocoder, trained with 227 Happiness Voice, 213 Anger Voice, 215 Sadness Voice. Source: HTS, STRAIGHT
  • Merlin DNN: Using Merlin Toolkit with DNN model, trained with 227 Happiness Voice, 213 Anger Voice, 215 Sadness Voice. Source: https://github.com/CSTR-Edinburgh/merlin

Emotion Type:

  • Happiness
  • Anger
  • Sadness
